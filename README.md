# Espaflor — База знаний

Репозиторий-база знаний для проектов и процессов.

## Структура

```
.
├── projects/          # Документация по отдельным проектам
├── people/            # Контекст по людям и командам
├── processes/         # Процессы, чек-листы, SOP
└── decisions/         # Лог решений (ADR)
```

## Как это работает

- Репозиторий — **источник правды** для базы знаний
- Пополняется из чатов с Claude (через GitHub MCP) и вручную через Obsidian/IDE
- История всех изменений видна в коммитах — ничего не теряется

## Конвенции

- Файлы в формате Markdown (`.md`)
- Имена файлов: `kebab-case.md`
- Новые заметки сопровождаются осмысленным commit message
